# BalanceIQ 🧠💸  
*A modern personal finance and budgeting SaaS application*

BalanceIQ helps users take control of their finances with real-time expense tracking, budgeting, and analytics.  
This project is built as a **full-stack web application**.

---

## ✨ Features
- 🔐 **Authentication & Security**
  - User registration & login
  - Email verification via OTP (Resend API)
  - Session handling with **HTTP-only cookies**
- 📊 **Finance Management (Planned)**
  - Add and categorize income & expenses
  - Set and track monthly budgets
  - Visual analytics dashboard
- 🚀 **Scalable Architecture**
  - REST API design
  - Modular and clean codebase

---

## 🛠 Tech Stack
**Frontend**
- Next.js (App Router)
- Tailwind CSS
- Zustand (for state management)

**Backend**
- Node.js + Express
- MongoDB + Mongoose
- JWT
- Resend API for OTP emails
